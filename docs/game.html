游戏

介绍
Minecraft是一款沙盒类电子游戏，中国版官方译为《我的世界》，又称《当个创世神》，开创者为马库斯·泊松（Notch）。游戏现由Mojang Studios维护，隶属于微软Xbox游戏工作室。 自开创伊始到延斯·伯根斯坦（Jeb）加入并负责开发之前，Minecraft几乎全部的开发工作由Notch完成。游戏音乐由丹尼尔·罗森费尔德和莉娜·雷恩创作；克里斯托弗·泽特斯特兰绘制了游戏中的画。游戏最初于2009年5月17日作为Classic版本发布，并于2011年11月18日发布正式版。至今，Minecraft的游戏平台已囊括移动设备和游戏主机。2014年11月6日，Minecraft及Mojang的全部资产被微软以25亿美元收购。Notch从此离开了Mojang，不再为Minecraft工作。截至2021年5月，Minecraft的销量已经突破2.38亿份，月活跃用户高达1.4亿。 Minecraft着重于让玩家探索、交互并改变一个动态生成1m³大小方块的世界。除了方块以外，环境中还包括动植物与物品。游戏的内容包括采集矿石，与敌对生物战斗，以及收集游戏中的各种资源来合成新的方块与工具。开放的游戏模式允许玩家在各类多人服务器或单人地图中创造建筑物与艺术作品。其他内容包括用于逻辑运算与远程控制的红石电路、矿车及轨道，以及称之为“下界”的神秘维度。最终，可以选择前往一个叫做“末地”的维度旅行，并击败末影龙。
Minecraft：Java版（Minecraft: Java Edition）是Minecraft的最初版本，由Mojang Studios为Windows、macOS与Linux开发。Notch于2009年5月10日开始游戏的开发，于2009年5月17日公开发布首个版本。完整的正式版游戏于2011年11月18日的MINECON 2011上发布。
基岩版（Bedrock Edition）（官方称为无副标题的Minecraft）是由Mojang Studios、Xbox Game Studios和SkyBox Labs开发的多平台版本Minecraft，基于与携带版一脉相承的游戏引擎基岩引擎。
Minecraft教育版（Minecraft Education）是Minecraft特别为教室使用而设计的教学版本。它由Mojang Studios和Xbox Game Studios共同开发，包含了让Minecraft在课堂里容易使用的特性。完整的游戏已在2016年11月1日发布。Beta版本的测试在2016年6月9日至11月1日间进行。 2018年8月20日，Mojang宣布将于2018年秋季教育版将支持iPad。微软表示，许可用户可以在9月开始使用。它已于2018年9月6日在App Store发布。 2019年3月27日，微软宣布教育版将由京东在中国经营。2022年1月，京东关闭中国教育版官方网站，停止与微软的合作及教育版的运营。关闭后进入该网站会被重定向并显示：“此活动已结束，您可参加京东云其他优惠活动”。 2020年6月26日，针对Google Play商店的兼容Chromebook推出了教育版公开测试版。完整游戏已于2020年8月7日发布到Chromebook的Google Play商店。
我的世界：中国版（Minecraft: China Edition），官方表述为《我的世界》中国版，是Minecraft Java版和基岩版的中国本地化版本，由Mojang Studios、Xbox Game Studios和网易联合开发。截至2022年10月已有6亿注册用户。 中国版主要包括手游版和端游版，提供基岩版和Java版游戏。中国版是网易开发的手游/端游启动器和启动器内所包含Minecraft游戏的总称，启动器同时具有多人游戏、组件管理等功能。

Minecraft Dungeons，又名我的世界：地下城，是一款动作/冒险角色扮演类电子游戏，由Mojang Studios、Xbox Game Studios和Double Eleven共同开发。它于2020年5月26日发布在PC、Nintendo Switch、PS4和Xbox One平台，并在2020年11月10日向后支持发布在Xbox Series X|S平台，之后于2021年9月22日发布在Steam平台。Windows 10，Windows 11和Xbox版本同样可以通过Xbox Game Pass进行游戏，也可以通过Xbox云游戏在手机端进行游戏。基础游戏价格为US$19.99（￥144，Microsoft Store为￥69，HK$154，€19.99，£16.74，AU$29.95)，英雄版（已停售）价格为US$29.99（￥214，Microsoft Store为￥106，HK$234，€29.99，£24.99，AU$39.95)，并包含额外的披风，宠物，以及觉醒丛林和悚伏寒冬两个DLC，普通版可通过购买英雄通行证（已停售）和购买带有四个DLC的第二个扩展包季票进行补齐（已停售）或者直接购买终极版DLC捆绑包，价格为¥144（US$19.99）。或者不需购买游戏本体，直接购买终极版，售价为¥284（US$39.99）。目前已推出其中六个DLC。1.12.0.0中增加了冒险通行证，其价格为￥28（US$3.99）。 目前游戏可在官网、Microsoft Store和Steam进行购买，中国大陆地区在游戏官网进行购买会出现Minecraft Dungeons未在该地区上市，将由网易公司引进代理的提示，但仍然可以用人民币结算正常进行购买。

Minecraft Legends，又名我的世界：传奇，是一款由Mojang Studios、Xbox Game Studios和Blackbird Interactive开发的即时动作战略游戏，发布于2023年4月18日。

Minecraft: Story Mode（我的世界：故事模式）是一个以沙盒电子游戏Minecraft为基础的，具有可点选性、叙事性、章节性的电子游戏。它在2015年10月同时在多个平台发布，同年12月16日在Windows 10发布，2016年1月21日在Wii U发布，2016年8月24日在Apple TV上发布，并且2017年8月22日在Nintendo Switch上发布。这游戏由Minecraft的开发者Mojang Studios与Telltale Games合作开发。游戏的音乐由美国加利福尼亚州一个以大气音乐为基础的乐队Anadel创作。该游戏继承Telltale以往游戏的故事形式，玩家的选择和行动对后面的故事元素有着重要的影响。 它的续集，Minecraft: Story Mode - Season Two，在2017年7月11日发布，以新的故事情节和人物为特色。 2019年5月31日，因为前Telltale Games公司的关闭，Minecraft: Story Mode被宣布不再受支持。它的服务器于2019年6月25日关闭，意味着游戏无法再下载。
Minecraft: Story Mode - Season Two（我的世界：故事模式 - 第二季）是一个以沙盒电子游戏Minecraft为基础的电子游戏，具有可点选性、叙事性与章节性，于2017年7月同时在多个平台发布，并于2018年11月6日在Nintendo Switch上发布。该游戏由Minecraft的开发者Mojang Studios与Telltale Games合作开发。游戏的音乐由美国加利福尼亚州一个以大气音乐为基础的乐队Anadel创作。该游戏继承Telltale以往游戏的故事形式，玩家的选择和行动对后面的故事元素有着重要的影响。 2019年5月31日，因为前Telltale Games公司关闭，Minecraft: Story Mode经宣布不再受支持，其服务器于2019年6月25日关闭，意味着游戏无法再下载。
